# TriviaQuizApp
四択式のクイズアプリです。６つのジャンルから一つを選択し、１０問問題を解きます。
Udemyで学習したことを元に一からオリジナルアプリの作成に挑戦しました。

#機能
最初の画面でcollectionViewを使いジャンルを表示しています。
問題のデータはfirestoreに格納しています。選ばれたジャンルにあるfirestoreのデータを読み込み、ランダムに四択の問題を表示しています。
